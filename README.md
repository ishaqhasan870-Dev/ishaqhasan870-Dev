## 🚀 Featured Project

### 🔐 [Social Authentication System using Django](https://github.com/ishaqhasan870-Dev/A-Web-Based-Approach-to-Social-Authentication-Systems-using-Django)

> A secure, scalable web application implementing modern authentication mechanisms.

A full-stack Django project that replaces vulnerable password-based logins with OAuth2 social authentication. Built with a focus on security, user experience, and modern best practices.

**Key Features:**
- ✅ OAuth2 Social Login (sign in via existing social media accounts)
- ✅ Multi-Factor Authentication (MFA)
- ✅ Role-Based Access Control (RBAC)
- ✅ Session Management & CSRF Protection
- ✅ Document sharing between authenticated users
- ✅ Friend search & connection system

**Tech:** `Python` · `Django` · `OAuth2` · `HTML/CSS/JS` · `SQLite`
